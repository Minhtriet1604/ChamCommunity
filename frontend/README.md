# Cham Community

ChamCommunity là một trang web dành cho cộng đồng người Chăm, cung cấp thông tin về văn hóa, tin tức và sự kiện của cộng đồng.

## Tính năng hiện có

### Trang chủ (index.html)
- Banner giới thiệu trang web
- Phần tin tức nổi bật
- Các sự kiện sắp diễn ra
- Video giới thiệu Kate 2025
- Biểu tượng mạng xã hội tích hợp
- Footer với thông tin liên hệ

### Tin tức (news.html)
- Danh sách tin tức với hình ảnh
- Phân trang
- Tìm kiếm tin tức
- Chi tiết tin tức (news-detail.html)
- Video sự kiện Kate 2025

### Lịch sự kiện (calendar.html)
- Hiển thị lịch tháng
- Đánh dấu các ngày có sự kiện
- Chi tiết sự kiện khi click vào ngày

### Thư viện ảnh (gallery.html)
- Hiển thị ảnh dạng lưới
- Phân loại ảnh theo chủ đề
- Chế độ xem ảnh phóng to
- Slider ảnh

### Diễn đàn (forum.html)
- Giao diện chính diễn đàn
- Danh sách chủ đề thảo luận
- Form đăng bài (đang phát triển)

### Văn hóa (culture.html)
- Giới thiệu văn hóa Chăm
- Các lễ hội truyền thống
- Phong tục tập quán
- Hình ảnh minh họa

### Liên hệ (contact.html)
- Form liên hệ
- Bản đồ địa điểm
- Thông tin liên lạc
- Kết nối mạng xã hội

### Tính năng chung
- Giao diện responsive trên mọi thiết bị
- Menu điều hướng thông minh
- Tích hợp mạng xã hội
- Hiệu ứng chuyển trang mượt mà

## Cấu trúc thư mục

```
frontend/
├── about.html          # Trang giới thiệu
├── calendar.html       # Trang lịch sự kiện
├── contact.html        # Trang liên hệ
├── culture.html        # Trang văn hóa
├── forum.html          # Trang diễn đàn
├── gallery.html        # Trang thư viện ảnh
├── index.html          # Trang chủ
├── news.html           # Trang tin tức
├── news-detail.html    # Trang chi tiết tin tức
├── assets/
│   ├── css/           # Thư mục chứa file CSS
│   │   └── style.css  # File CSS chính
│   └── js/            # Thư mục chứa file JavaScript
│       └── script.js  # File JavaScript chính
└── anh/               # Thư mục chứa hình ảnh
```

## Kế hoạch phát triển

### Giai đoạn 1 - Hệ thống người dùng (Đang thực hiện)
- Hệ thống đăng ký/đăng nhập
  - Form đăng ký với xác thực email
  - Đăng nhập bằng tài khoản mạng xã hội
  - Quản lý thông tin cá nhân
  - Khôi phục mật khẩu
- Phân quyền người dùng
  - Quản trị viên
  - Biên tập viên
  - Thành viên
  - Khách

### Giai đoạn 2 - Tương tác người dùng (Kế hoạch)
- Chức năng bình luận
  - Bình luận tin tức
  - Thảo luận trong diễn đàn
- Chia sẻ bài viết
  - Chia sẻ lên mạng xã hội
  - Gửi qua email
- Đánh giá và phản hồi
  - Like/Dislike bài viết
  - Báo cáo nội dung không phù hợp

### Giai đoạn 3 - Nâng cao trải nghiệm (Kế hoạch)
- Hỗ trợ đa ngôn ngữ
  - Tiếng Việt
  - Tiếng Chăm
  - Tiếng Anh
- Tối ưu hóa hiệu suất
  - Lazy loading hình ảnh
  - Nén và tối ưu tài nguyên
  - Caching
- Tính năng tìm kiếm nâng cao
  - Tìm kiếm theo nhiều tiêu chí
  - Gợi ý tìm kiếm
  - Lọc kết quả

### Giai đoạn 4 - Mở rộng tính năng (Kế hoạch)
- Tích hợp thanh toán
  - Ủng hộ dự án
  - Mua sắm sản phẩm văn hóa
- Hệ thống thông báo
  - Email thông báo
  - Push notification
